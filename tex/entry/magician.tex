\item[Magician,]
\entlbl{magician}

\grc{φάρμακος}
\index[grc]{φαρμακος@\grc{φάρμακος}}
(\textit{pharmakos}):
Liddell denotes ``\emph{a poisoner}, \emph{sorcerer}, \emph{magician},'' to which both Thayer and Gingrich agree. Danker defines \emph{magician} as ``an expert in manipulation through occult means.'' See \entref{witchcraft}.
Found in Rev 21:8, 22:15.
